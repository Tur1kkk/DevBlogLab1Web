### Завдання до роботи
**Тема:** Розробка статичного інтерфейсу Web-додатка.
**Варіант:** Блог з коментарями.

**Мета:** Ознайомитись із засобами мов HTML5 та CSS3 та навчитись застосовувати бібліотеку Twitter Bootstrap для побудови адаптивного Web-інтерфейсу користувача.

**Опис проєкту:**
В межах роботи реалізовано п'ять обов'язкових статичних сторінок:
1. **Головна (Робоча сторінка):** Стрічка повідомлень блогу з розділом коментарів.
2. **Профіль користувача:** Дані користувача у табличному вигляді.
3. **Реєстрація:** Форма реєстрації (ім'я, email, стать, дата народження).
4. **Вхід:** Форма авторизації (email, пароль).
5. **Про додаток:** Опис функціоналу та емблема проєкту.

Усі сторінки мають єдиний стиль (header, menu, main content, footer) та адаптовані для коректного відображення на мобільних пристроях і планшетах.
