### Завдання до роботи
**Тема:** Розробка функціональності Web-додатка мовою Javascript.
**Варіант:** Блог з коментарями.

**Мета:** Ознайомитись із засобами мови Javascript та навчитись їх застосовувати для побудови Web-інтерфейсу користувача.

**Опис проєкту:**
В межах роботи реалізовано динамічну функціональність для статичних сторінок Web-додатку із використанням шаблону MVC (Model-View-Controller) для головної сторінки. Сторонні Web-фреймворки (ReactJS, AngularJS, VueJS) не використовувалися. 

Реалізовано наступний функціонал:
* **Головна (Робоча сторінка):** Динамічне виведення списку повідомлень, додавання нових постів та коментарів, а також вилучення власних записів. Логіка побудована за шаблоном MVC.
* **Сховище даних:** Використано структури даних Javascript та `localStorage` для збереження стану додатку.
* **Авторизація та Реєстрація:** Додано логіку перевірки даних та збереження поточного користувача.
* **Профіль користувача:** Реалізовано відображення персональних даних з `localStorage` та функцію їх редагування.
